# Example 2 Flow Chart

- **Task:** Take input of a salary. If the salary is greater than 10,000, add a bonus of 2000; otherwise, add a bonus of 1000.
- **Flow Chart:**
   - (Start)
   - (Input Salary)
   - (Condition: Salary > 10000?)
   - (Processing: Salary = Salary + 2000 or Salary = Salary + 1000)
   - (Output Salary)
   - (Stop)

# Example 2 Pseudocode

```
Start
Input Salary
if Salary > 10000:
    Salary = Salary + 2000
else:
    Salary = Salary + 1000
Output Salary
Exit
```
